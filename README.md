# AGM
Árvore Geradora Máxima

Aplicação de interface simples que simula um algoritmo de Kruskal para resolver problema de uma árvore geradora máxima, onde a soma dos pesos das arestas é o maior valor dentre as possibilidades descritas no grafo.

O programa recebe um arquivo .txt que segue um padrão, esse arquivo representa o grafo e após a execução do algoritmo ele exibe o peso máximo desse grafo.
